# ISYS3004 Assessment 1 — Sample Portfolio

A reference implementation of the Assessment 1 portfolio for **ISYS3004 Business Web and Mobile Technologies** at Curtin University.

**Live site:** https://michael-borck.github.io/isys3004-a1-sample/

---

## What this is

This is a worked example showing what a solid Assessment 1 submission looks like — not a perfect or advanced site, but one that meets the brief professionally and demonstrates genuine understanding of the concepts covered in Modules 1–5.

The site is intentionally simple. Every line of HTML, CSS, and JavaScript is something a student in this unit could have written, and every decision has a reason that can be explained in a viva.

## What's in this repo

| File / Folder | Purpose |

| `docs/validation/` | Lighthouse and W3C validation reports |

## For students

Start with the [live site](https://michael-borck.github.io/isys3004-a1-sample/), then read [CODE-GUIDE.md](CODE-GUIDE.md). The guide explains the ideas behind the code — how HTML, CSS, and JavaScript connect, what selectors do, and how specific features like dark mode and the API fetch work. It also lists the kinds of questions you may be asked in the viva.
