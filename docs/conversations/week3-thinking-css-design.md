## What a real thinking chat covers

A Week 3 thinking chat would explore:

- What colour scheme to use and why — not just "what looks nice" but what feels right for a professional portfolio
- What CSS layout approach to use (Flexbox vs Grid, and when to use each)
- What "mobile-first" means and how it affects how you write media queries
- How CSS custom properties (variables) work and whether to use them
- What the navigation should look like at different screen sizes
- Whether to use a CSS reset and what it does

## Prompt examples from this week

> "I want a professional but personal colour scheme for my portfolio. I'm studying business and technology — not trying to look like a designer, but not wanting to look generic either. Help me think through colour choices and why they work."

> "Explain the difference between writing CSS mobile-first versus desktop-first. Which should I use and why?"

> "What are CSS custom properties and are they worth using for a project this size?"
