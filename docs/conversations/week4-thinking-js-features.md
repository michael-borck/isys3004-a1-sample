## What a real thinking chat covers

A Week 4 thinking chat would explore:

- What JavaScript features would actually be useful for a portfolio (not just technically impressive)
- How the hamburger mobile nav would work — the mechanism, not the code
- How a dark/light theme toggle works and how to persist the preference
- What `addEventListener` does and why it's better than inline `onclick` attributes
- What `localStorage` is and what it's appropriate for
- What `aria-expanded` is and why it needs to be updated when the nav opens

## Prompt examples from this week

> "I need two JavaScript features for my portfolio. The hamburger nav is obvious. What else would actually be useful — not just gimmicky — for a portfolio site at this level?"

> "Before I write any code, explain how a dark/light mode toggle would work mechanically. How does JavaScript tell the CSS to switch themes?"

> "What is localStorage and how is it different from a cookie? Is it appropriate for saving a theme preference?"
