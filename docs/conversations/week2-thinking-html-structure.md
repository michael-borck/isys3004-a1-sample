## What a real thinking chat covers

A thinking chat for Week 2 HTML structure would typically include:

- Questions about which semantic HTML elements to use for each part of the page
- Asking what the difference is between `<section>` and `<article>` and `<div>`
- Exploring what pages the portfolio needs and what goes on each one
- Asking about file structure — where do HTML, CSS, and JS files go?
- Questions about `<head>` content — what meta tags are needed and why
- Clarifying what "separation of concerns" means in practice

## What makes a good thinking chat

- Multiple turns (not a single prompt and done)
- Questions that get more specific as the conversation develops
- Pushing back on or questioning AI responses
- Asking "why" not just "what"
- Ending with a clear picture of what you're going to build

## Prompt examples from this week

> "I'm building a portfolio website for a web development unit. Before I start coding, help me think through the HTML structure I'll need. What pages should I have and what should go on each one?"

> "You suggested using `<section>` for the skills area — why that instead of `<div>`? What's the actual difference?"

> "What goes in the `<head>` tag and which meta tags are actually important for a portfolio site?"
